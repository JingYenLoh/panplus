Class: ContextBridge

ContextBridge(func, eventHandle, eventHandler)

new ContextBridge(func, eventHandle, eventHandler)

ContextBridge class for setting up linkages between the isolated code environment and the actual code environment of the user
Parameters:
Name Type Description
func function the function to be injected into the page.
eventHandle string the string used for triggering & detecting events for this particular bridge
eventHandler function the function called whenever the eventHandle is triggered. This function takes in 1 parameter.

close()

Clean-up, remove scripts and event listeners. This is automatically handled if you used request or exec. However, if you used connect (which establishes a connection), then you may want to call close() to terminate the connection.
