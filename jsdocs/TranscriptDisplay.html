Class: TranscriptDisplay

TranscriptDisplay(settings)

TranscriptDisplay, includes code that will load the transcript, then inject the subtitles as cues into the first video's texttrack. If there is a second video, then when the cue is reached, the subtitle will be injected into the second video. This is to circumvent the issue of the variable offset between the 1st and 2nd video currentTime.
